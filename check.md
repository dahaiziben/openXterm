# OpenXterm 验收检查清单

## 基础功能
- [ ] 打开软件显示连接列表
- [ ] 新建连接可以填写主机、端口、用户名、密码
- [ ] 密码输入框有小眼睛切换显示/隐藏
- [ ] 保存连接后再次打开，数据持久化
- [ ] 点击连接按钮，启动 ssh.exe 并自动填入密码
- [ ] 支持端口转发参数（-L、-R、-D）
- [ ] 支持跳板机（-J）配置

## 命令解析
- [ ] 粘贴 ssh -L 3306:127.0.0.1:3306 user@host -J jump@jump_host 自动解析
- [ ] 自动提取主机、端口、用户名、转发参数、跳板机

## 密码管理
- [ ] 密码加密存储在本地
- [ ] 小眼睛可以看清密码
- [ ] 保存的密码下次打开仍可用

## 打包部署
- [ ] 可生成单个 exe 安装包
- [ ] 在 Win10 全新机器上安装后可用
